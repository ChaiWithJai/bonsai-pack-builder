## Source claims we should keep stable

- Deep Agents provides planning, subagents, file systems, memory, tools, and human
  approval controls.
- HashiCorp RFCs are used to create clarity before a change affects many people.
- Nomad Pack packages repeatable workload patterns as packs and registries.
- Packer makes builds reproducible through declared templates and ordered stages.
- PrismML's public Bonsai materials frame the model family around 1-bit inference
  and intelligence density.
- The Bonsai-demo repo is the contribution target for community benchmark reports.
